# Fight Song Genome

An interactive visualization created for a university data visualization competition.  
Explore how college fight songs differ in tempo, duration, lyrical tropes, and conference style.

Live site: https://guinanguo.github.io/fight-song-viz/

Data: Fight Songs dataset (FiveThirtyEight) + derived metrics.

## Modules

- Hero Map: animated U.S. map with school locations and hub-centered rays.
- Geographic Distribution: static map for precise spatial reference.
- Conference Fingerprints: radar charts summarizing conference-level traits.
- Tempo vs Duration: scatter plot of energy and length profiles.
- Trope DNA Matrix: grid showing lyrical tropes by school.
- Similarity Network: graph linking schools with similar musical features.
- Multi-Dimensional Analysis: parallel coordinates across multiple metrics.
- Summary Poster: compact highlights with featured school spotlight.
